# FMCG-Sales-Analysis
Exploratory Data Analysis on FMCG Sales, Marketing and Profitability data using Python and Pandas.
## Insights Covered
- Region wise total profit
- Top 5 best selling product categories
- Marketing spend vs profit relationship
- Year wise revenue trend
- Sales channel performance
- Most profitable promotion type
- Brand wise profit margin analysis

## Tools Used
Python, Pandas, Matplotlib

## Dataset
FMCG Sales Marketing and Profitability 2023-2025 dataset from Kaggle
